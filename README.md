# DevPulse API

A collaborative backend platform for software teams to report bugs, suggest features, and manage issue workflows.

## Live URL

- Backend Live: https://dev-pulse-black.vercel.app/
- GitHub Repository: https://github.com/shuvoroy10/DevPulse

---

## Features

- User authentication with JWT
- Role-based authorization
- Create, update, and delete issues
- Contributor and maintainer roles
- Dynamic issue filtering and sorting
- Secure password hashing with bcrypt
- PostgreSQL database with raw SQL queries
- Modular Express.js architecture

---

## Technology Stack

- Node.js
- Express.js
- TypeScript
- PostgreSQL
- pg
- bcrypt
- jsonwebtoken
